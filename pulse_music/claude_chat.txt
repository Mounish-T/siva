Dart core layer (7 files)
lib/core/config/app_config.dart — central spot for API keys. Reads them from --dart-define at compile time so secrets never live in source. Has two booleans (jamendoConfigured, supabaseConfigured) so the rest of the app can degrade gracefully when keys are missing.

lib/core/services/jamendo_service.dart — the HTTP client that talks to api.jamendo.com/v3.0. Four methods: trending(), latest(), byTag(genre), search(query). lib/core/services/player_service.dart — the audio engine wrapper. Singleton holding one AudioPlayer. lib/core/services/auth_service.dart — Supabase auth wrapper. Methods for signUp, signIn, signOut. Guarded so it throws a clear error if you forgot to configure Supabase, instead of NPE-ing.

lib/core/services/providers.dart — all the Riverpod providers in one file. Bridges services to UI: trendingTracksProvider, latestTracksProvider, searchResultsProvider, plus stream providers for player state, and a FavoritesNotifier for the like button.
